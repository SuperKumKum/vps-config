Formation Technofutur C#/.NET

C# est le langage de programmation phare de Microsoft. Il est utilisé par un nombre important de professionnels et permet de réaliser tout type d'applications (Web, client lourd, Service Web, …).
La formation a pour but de présenter les bases du ce langage et d'expliquer l'environnement global sur lequel s'appuie le C#.

Avis

La formation m'a permis d'apprendre un autre langage de programmation qui est orienté objet autre que le Java que j'ai pu apprendre durant mon cursus à l'EPHEC. J'ai pu découvrir le langage de Microsft qui est puissant. Néanmoins, la programmation n'étant pas ma matière de prélidection, une semaine de formation était un peu trop court pour que je puisse comprendre suffisament la matière même si celle-ci est fort proche du Java.

Heures totales effectuées : ~ 40 heures
Heures valorisables : ~ 10 heures
Preuves : Certification
