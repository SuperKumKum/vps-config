Formation Cisco Meraki

Cisco Meraki est une solution complète de Cloud Networking couvrant les communications sans fil, les switchs, la sécurité, l'optimisation WAN et la gestion de terminaux mobiles, le tout géré de façon centralisée depuis le Web au moyen d'un tableau de bord unique. Meraki intègre les services matériels, logiciels et Cloud. Cette architecture réseau Cloud simple d'emploi permet de résoudre de nouveaux problèmes métiers et de réduire les coûts d'exploitation, tout en positionnant votre société en tant qu'expert dans le monde ultra-compétitif, nouveau et excitant du Cloud Networking.
Cette formation a pour but de présenter le produit et a former pour une configuration basique via l'interface web de Meraki.

Avis

Cette formation m'a été proposée par LogicalTIC, la société dans laquelle j'ai effectué mon stage.
Elle m'a permis d'en apprendre plus sur la gamme "Meraki" de Cisco étant une gamme plus facile d'accès et moins onéreuse pour les PME. J'ai pu également apprendre comment configurer la suite d'outils "Meraki" via le Cloud que ce soit Switch, Routeur ou encore Access Point.
Cette formation était très intéressant de mon point de vue, car c'était la première fois que je voyais une configuration autre que par le biais de la console comme j'ai appris durant mon cursus scolaire.

Heures totales effectuées : ~ 8 heures
Heures valorisables : ~ 8 heures
